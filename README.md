# MBTI 性格测试

一个基于 React + Vite 的 MBTI 性格测试应用，包含 93 道标准测试题目。

## 功能特点

### ✨ 核心功能
- **完整测试流程**：93 道精心设计的 MBTI 测试题目
- **实时进度显示**：清晰的进度条和题号提示
- **支持返回上一题**：可以修改之前的答案
- **详细结果分析**：展示四个维度的详细得分和百分比
- **示例结果展示**：虚拟人物"李明"的测试结果示例，帮助用户了解结果页面
- **键盘快捷键**：支持 A/B 或 1/2 快速选择，← 返回上一题
- **数据持久化**：自动保存答题进度，刷新页面不丢失
- **历史记录**：保存所有测试结果，可随时查看历史记录
- **结果分享**：下载结果图片或复制文本，方便分享
- **移动端优化**：响应式设计，完美支持手机和平板

### 📊 结果展示
测试完成后，会显示：
1. **MBTI 类型**：如 INTJ、ENFP 等 16 种性格类型之一
2. **四个维度的详细分析**：
   - **E/I (外向/内向)**：能量来源
   - **S/N (实感/直觉)**：认知方式
   - **T/F (思考/情感)**：判断方式
   - **J/P (判断/感知)**：生活方式

3. **每个维度显示**：
   - 具体得分（选择该选项的题目数量）
   - 百分比（该选项占该维度总题数的比例）
   - 可视化进度条
   - 倾向性说明

## 计分逻辑

### 数据收集
- 用户每次选择会记录对应的维度值（E/I/S/N/T/F/J/P）
- 所有答案存储在数组中

### 结果计算
1. **统计各维度得分**：计算每个字母被选择的次数
2. **计算百分比**：
   - 每个维度的总题数 = 该维度两个选项的得分之和
   - 百分比 = (该选项得分 / 维度总题数) × 100%
3. **确定类型**：每个维度选择得分较高的一方
4. **生成 MBTI 类型**：组合四个维度的结果

### 示例
假设某用户的答案统计：
- E: 12 题，I: 11 题 → 外向 (E) 52.2%
- S: 15 题，N: 8 题 → 实感 (S) 65.2%
- T: 14 题，F: 9 题 → 思考 (T) 60.9%
- J: 16 题，P: 8 题 → 判断 (J) 66.7%

最终结果：**ESTJ**

## 🎯 使用方法

### 首次使用
1. **访问** http://localhost:5173
2. **欢迎页面**：
   - 点击"🚀 开始测试"开始新测试
   - 点击"👁️ 查看示例结果"查看虚拟人物的测试结果
3. **答题过程**：
   - 使用鼠标点击选项 A 或 B
   - 或使用键盘快捷键：A/1 选择 A，B/2 选择 B，← 返回上一题
   - 进度自动保存，刷新页面可继续
4. **查看结果**：
   - 自动显示详细的性格分析
   - 点击"📥 下载结果图片"保存结果
   - 点击"📋 复制结果文本"分享结果
5. **历史记录**：
   - 所有测试结果自动保存在本地
   - 在欢迎页点击"📊 查看历史记录"查看所有测试

### 继续上次测试
如果有未完成的测试，欢迎页会显示"▶️ 继续上次测试"按钮

## 开发说明

### 修改题目
编辑 `src/data/questions.json` 文件，格式如下：

```json
{
  "question": "问题内容",
  "choice_a": {
    "value": "E",
    "text": "选项A的文本"
  },
  "choice_b": {
    "value": "I",
    "text": "选项B的文本"
  }
}
```

### 自定义样式
- 修改 `src/App.css` 调整整体布局
- 修改 `src/components/TestPage.css` 调整测试页面样式
- 修改 `src/components/ResultPage.css` 调整结果页面样式
