## 🎯 5 个选项说明

### 1. 😄 完全A（权重 1.0）
- **含义**：完全符合选项 A 的描述
- **快捷键**：按 `1`
- **得分**：A 得 1.0 分，B 得 0.0 分

### 2. 🙂 偏向A（权重 0.75）
- **含义**：比较符合选项 A 的描述
- **快捷键**：按 `2`
- **得分**：A 得 0.75 分，B 得 0.25 分

### 3. 😐 中立（权重 0.5）
- **含义**：两个选项都有一定符合，难以判断
- **快捷键**：按 `3`
- **得分**：A 得 0.5 分，B 得 0.5 分

### 4. 🙁 偏向B（权重 0.25）
- **含义**：比较符合选项 B 的描述
- **快捷键**：按 `4`
- **得分**：A 得 0.25 分，B 得 0.75 分

### 5. 😞 完全B（权重 0.0）
- **含义**：完全符合选项 B 的描述
- **快捷键**：按 `5`
- **得分**：A 得 0.0 分，B 得 1.0 分

## 💡 使用建议

### 如何选择合适的选项？

#### 情况 1：非常确定
- 如果你**完全认同**某个选项 → 选择 **😄 完全A** 或 **😞 完全B**
- 例如："你喜欢独处吗？" → 如果你非常喜欢独处，选择 **😞 完全B**

#### 情况 2：比较倾向
- 如果你**大部分时候**符合某个选项 → 选择 **🙂 偏向A** 或 **🙁 偏向B**
- 例如："你喜欢计划吗？" → 如果你通常喜欢计划但偶尔也会随性，选择 **🙂 偏向A**

#### 情况 3：难以判断
- 如果你**两者都有**或**无法确定** → 选择 **😐 中立**
- 例如："你更关注细节还是整体？" → 如果你觉得两者都重要，选择 **😐 中立**

## 📊 计分原理

### 权重系统

每个选项都有一个权重值（0.0 - 1.0），表示你对某个维度的倾向程度：

| 选项 | 权重 | A 得分 | B 得分 | 说明 |
|------|------|--------|--------|------|
| 😄 完全A | 1.0 | 1.0 | 0.0 | 100% 倾向 A |
| 🙂 偏向A | 0.75 | 0.75 | 0.25 | 75% 倾向 A，25% 倾向 B |
| 😐 中立 | 0.5 | 0.5 | 0.5 | 50% 倾向 A，50% 倾向 B |
| 🙁 偏向B | 0.25 | 0.25 | 0.75 | 25% 倾向 A，75% 倾向 B |
| 😞 完全B | 0.0 | 0.0 | 1.0 | 100% 倾向 B |

### 计算示例

假设 E/I 维度有 23 题，你的选择如下：
- 10 题选择 **😄 完全E**（权重 1.0）
- 5 题选择 **🙂 偏向E**（权重 0.75）
- 3 题选择 **😐 中立**（权重 0.5）
- 3 题选择 **🙁 偏向I**（权重 0.25）
- 2 题选择 **😞 完全I**（权重 0.0）

**计算过程**：
```
E 得分 = 10×1.0 + 5×0.75 + 3×0.5 + 3×0.25 + 2×0.0
       = 10 + 3.75 + 1.5 + 0.75 + 0
       = 16.0 分

I 得分 = 10×0.0 + 5×0.25 + 3×0.5 + 3×0.75 + 2×1.0
       = 0 + 1.25 + 1.5 + 2.25 + 2
       = 7.0 分

总分 = 16.0 + 7.0 = 23.0 分

E 百分比 = (16.0 / 23.0) × 100% = 69.6%
I 百分比 = (7.0 / 23.0) × 100% = 30.4%
```

**结果**：你倾向于 **外向 (E)**，占 69.6%

## ✨ 优势

### 相比传统二选一：
1. **更精确**：5 个选项比 2 个选项更能反映真实倾向
2. **更灵活**：允许表达"两者都有"的情况
3. **更科学**：权重系统使结果更加准确
4. **更友好**：减少"强迫选择"的不适感

### 向后兼容：
- 旧数据自动识别（没有 weight 字段时默认为 1.0）
- 用户无需重新测试
