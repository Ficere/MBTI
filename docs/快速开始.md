# 快速开始

## 🚀 网页版

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:5173

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

---

## 📱 小程序版

### 进入小程序目录

```bash
cd miniapp
```

### 安装依赖

```bash
npm install
```

### 开发微信小程序

```bash
npm run dev:weapp
```

### 在微信开发者工具中打开

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniapp/dist` 目录
4. 填入你的小程序 AppID（或使用测试号）

### 构建微信小程序

```bash
npm run build:weapp
```

## 📝 使用说明

### 网页版答题流程

1. **欢迎页面**
   - 点击"🚀 开始测试"开始新测试
   - 点击"👁️ 查看示例结果"查看示例
   - 扫描二维码访问网页版

2. **答题页面**（5 选项）
   - 查看选项 A 和选项 B 的描述
   - 选择你的倾向程度：
     - 😄 完全A（按 1）
     - 🙂 偏向A（按 2）
     - 😐 中立（按 3）
     - 🙁 偏向B（按 4）
     - 😞 完全B（按 5）
   - 按 ← 返回上一题

3. **结果页面**
   - 查看详细的性格分析
   - 下载结果图片
   - 复制结果文本
   - 生成分享二维码

4. **历史记录**
   - 查看所有测试记录
   - 删除单条记录
   - 清空所有记录

## 🔧 常见问题

### Q: 如何修改题目？

编辑 `src/data/questions.json` 文件。

### Q: 如何修改选项权重？

编辑 `src/constants/options.js` 文件。

### Q: 如何修改 MBTI 类型描述？

编辑 `src/constants/mbti.js` 文件中的 `TYPE_DESCRIPTIONS` 对象。

### Q: 如何部署到服务器？

参考 [部署说明](部署说明.md)。

### Q: 如何开发小程序？

参考 [小程序开发指南](小程序开发指南.md)。

### Q: 旧数据会丢失吗？

不会。新版本自动兼容旧数据格式（没有 weight 字段时默认为 1.0）。
